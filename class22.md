## 1. What is Terraform?
An Infrastructure as Code (IaC) tool.

Used to:
- Provision cloud resources
- Automate infrastructure
- Maintain consistency

Supports:
AWS, Azure, GCP, and more.

---

## 2. Why Infrastructure as Code?
- Repeatable setups
- Version-controlled infra
- Reduced manual errors
- Faster provisioning

---

## 3. Terraform Lifecycle

### terraform init
Initializes project and downloads providers.

### terraform plan
Shows what changes will occur.

### terraform apply
Applies changes.

### terraform destroy
Removes resources.

---

## 4. Advantages
- Speed
- Consistency
- Version control
- Multi-cloud support
