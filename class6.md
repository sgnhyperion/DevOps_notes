# Introduction to Docker & Containers

* Docker is an open platform to **build, ship, and run applications**
* It separates **applications from infrastructure**, making delivery faster
* Infrastructure can be managed the same way as applications

---

# Containers

* Containers package an app with **libraries and dependencies**
* Ensures the app runs **consistently across environments**
* Lightweight compared to virtual machines
* Share host OS instead of running a full OS
